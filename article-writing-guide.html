1️⃣ 文章撰寫 SOP（5 步法）

  1. 選題：從 GSC 找「曝光高但 CTR 低」的關鍵字 → 這代表 Google 願意露你，但消費者不點 → 寫一篇強標題的文章可立即拿流量
  2. 競品偵察：Google 搜該關鍵字看前 3 名（婚禮路線：良九、新娘物語、小殼手作；商業路線：高星活動、JIS、U-Design），看他們講什麼、缺什麼
  3. 找獨特角度：滿屋的差異化是「主題包套式整合」「不只租賃還含設計」「全省服務」「B2B+B2C 雙線經驗」
  4. 結構化撰寫：照下方「標準結構」寫，每段 100-200 字、總長 1,000-1,500 字
  5. SEO 健檢：寫完用 Rank Math 內建分析（綠燈 ≥80 分）+ 提交給軍師抽檢 Schema/Meta

2️⃣ 標準文章結構（直接拿來填）

H1（1 個）
含「主關鍵字 + 主訴求 + 品牌差異」例：「【婚禮佈置完整指南】從 5 萬到 50 萬的場景設計與費用規劃」
第一段 100 字
主關鍵字出現 2 次、自然帶到「滿屋設計」一次、結尾留問題鉤子
H2（4-6 個）
每個 H2 後面 200-300 字段落，從「問題 / 方案 / 案例 / 行動」鋪陳
H3（每 H2 配 0-2 個）
細節展開、可用 list / 表格輔助
圖片
每 300-500 字一張，alt 必填獨特描述（避免重複）
內部連結
每 800 字 1-2 個，鏈到風格分類頁或服務落地頁
FAQ 區塊
結尾 3-5 個問答，用 Rank Math FAQ block 寫（會自動上 FAQPage Schema 拿 Rich Snippet）
CTA
「立即免費諮詢／LINE 加好友／加入 Email 名單」三選一

3️⃣ SEO Meta 撰寫規範

Title（SEO 標題，55-60 字）

[主關鍵字]｜[次關鍵字/賣點]｜滿屋設計

✅ 範例：「聖誕新年節慶佈置｜社區・商場・企業空間｜滿屋設計」（5/14 已用此格式）

Meta Description（120-150 字）

[做什麼] + [覆蓋範圍] + [差異化] + [CTA]

✅ 範例：「滿屋設計承接社區、商辦、商場、企業大廳的聖誕與新年節慶佈置。大型裝置藝術、聖誕樹主題打卡區、跨年倒數氛圍規劃，讓你的空間在節慶成為最熱鬧的打卡地標。」（5/14 已用此格式）

⚠️ 避免：機械化模板（如「我們是專業的 XX，提供 XX 服務」）。Google 的 Helpful Content System 會降權這類內容。

5️⃣ ChatGPT/Claude Prompt 模板（複製即用）

你是滿屋設計（joyhouse-rental.com）的專業文案編輯。 請以 SEO 友善方式撰寫一篇文章： 【主題】：[填入主題] 【主關鍵字】：[填入 1 個主關鍵字] 【次關鍵字】：[填入 2-3 個次關鍵字] 【目標讀者】：[新人 / 企業 HR / 社區管委會 / 商場主管 擇一] 【字數】：1,200 字 【語氣】：專業但溫暖，B2B 與 B2C 之間的平衡 【結構要求】： 1. H1 含主關鍵字 + 痛點/欲望 2. 第一段 100 字內含主關鍵字 2 次 3. 4-6 個 H2 段落（問題-方案-案例-行動的鋪陳） 4. 結尾 3 個 FAQ 問答 5. 文末 CTA 引導 LINE 加好友或免費諮詢 【SEO Meta】： - Title（55-60 字）：[主關鍵字]｜[賣點]｜滿屋設計 - Meta Description（120-150 字）：[做什麼] + [覆蓋範圍] + [差異化] + [CTA] 【滿屋設計品牌脈絡】： - 承接全省婚禮、商業活動、企業春酒、社區節慶、校園活動 - 不只租賃，含完整設計與場控 - 直接競爭：婚禮路線（良九、新娘物語、小殼手作）/ 商業路線（高星活動、JIS） - 不寫機械化模板、不抄競品 請輸出：文章本體 + Title + Meta Description。

6️⃣ 寫作 checklist

寫作前（給文案）

  • 主關鍵字確認過 GSC + Google Trends
  • 競品 Top 3 文章都看過、找到 1 個獨特角度
  • 確認文章對應的「服務落地頁」或「分類頁」內部連結目標
  • 準備 3-5 張原創或合法授權圖片（含 alt 描述）

寫完前審稿（給軍師 / 主編）

  • H1 含主關鍵字、不超過 60 字
  • Title (SEO) 含主關鍵字 + 「滿屋設計」
  • Meta Description 120-150 字、含 CTA
  • 第一段 100 字內主關鍵字出現 2 次
  • 內部連結至少 2 個（風格分類頁 / 服務落地頁）
  • FAQ block 3-5 題（拿 Rich Snippet）
  • 圖片 alt 全部填寫且不重複
  • Rank Math SEO 分數 ≥ 80（綠燈）

7️⃣ 滿屋設計禁忌與守則

禁忌原因
❌ 寫「我們是專業的」這類空話Google Helpful Content System 降權，且讀者不買單
❌ 抄競品（良九、新娘物語、迪爾設計等）會被 Google 偵測為重複內容
❌ 把迪爾設計、520 氣球佈置列為對手非同主戰場（他們做氣球派對）
❌ Meta Description 用機械模板CTR 慘淡，等於白做
❌ 文章內塞太多關鍵字（密度 >3%）Google 視為過度優化會降權
✅ 第一人稱「滿屋設計」（不要「本公司」）更有溫度、品牌記憶度高
✅ 引用真實案例（含客戶類型 + 場地 + 預算範圍）建立 E-E-A-T（經驗、專業、權威、可信度）
✅ 結尾留 CTA 鉤子（LINE / 免費諮詢）把流量轉換為詢價
